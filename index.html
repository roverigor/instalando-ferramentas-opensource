🚨 Troubleshooting Comum

🌐 Problemas de DNS e Acesso

Problema: "Site não encontrado" (ERR_NAME_NOT_RESOLVED)

Sintomas:

  • Navegador mostra erro "não foi possível encontrar o servidor"
  • URL não abre

Soluções:

Solução 1: Aguardar Propagação

  • Tempo: 2-24 horas (geralmente 5-30 min)
  • Ação: Aguardar e tentar novamente

Solução 2: Verificar DNS no Cloudflare

  1. Cloudflare → DNS → Records
  2. Verificar:
    • Registro A para @ → IP da VPS ✓
    • Registro A para * → IP da VPS ✓
    • Proxy status: Proxied (nuvem laranja) ✓

🔒 Problemas de SSL/HTTPS

Problema: "Sua conexão não é privada"

Sintomas:

  • Navegador bloqueia acesso
  • Certificado SSL inválido

Soluções:

  1. Aguardar 5-15 minutos - Certificado sendo gerado
  2. Verificar modo SSL: Cloudflare → SSL/TLS → "Full"
  3. Ativar HTTPS: "Always Use HTTPS" → ON

🐳 Problemas de Containers

Container em Loop (Restart Constante)

Sintomas: Container com status "restarting"

Como Resolver:

  1. Ver Logs: Portainer → Containers → Logs
  2. Identificar erro (mensagens em vermelho)
  3. Verificar variáveis na stack
  4. Verificar recursos (RAM/CPU)

💾 Problemas de Recursos

RAM Insuficiente

Sintomas:

  • Containers parando sozinhos
  • VPS lenta
  • Erro "OOM" nos logs

Solução Rápida: Aumentar Swap

# Via SSH
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

# Tornar permanente
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

Solução Definitiva: Upgrade da VPS (2GB → 4GB RAM)

📝 Configuração Passo a Passo

1. Criar Conta

  1. Acesse: cloudflare.com
  2. Clique em "Sign Up"
  3. Digite email e senha
  4. Verifique seu email

2. Adicionar Domínio

  1. Dashboard → "Add a Site"
  2. Digite: seudominio.com
  3. Escolha plano: Free
  4. Aguarde scan de DNS (30-60s)

3. Configurar Nameservers

Anote os nameservers do Cloudflare:

alex.ns.cloudflare.com
erin.ns.cloudflare.com

Altere no registrador do domínio:

  1. Acesse painel do Registro.br/Hostinger
  2. Vá em "Nameservers" ou "DNS"
  3. Remova nameservers antigos
  4. Adicione os 2 do Cloudflare
  5. Salve alterações
  6. Aguarde 2-24h para propagar

4. Configurar Registros DNS

Type Name Content Proxy
A @ SEU_IP_VPS ☁️ Proxied
A * SEU_IP_VPS ☁️ Proxied

5. Configurar SSL/TLS

  1. SSL/TLS → Overview → Modo: "Full"
  2. Edge Certificates → "Always Use HTTPS": ON
  3. Automatic HTTPS Rewrites: ON
  4. Minimum TLS Version: TLS 1.2

6. Otimizações Recomendadas

  • Brotli Compression: ON
  • Auto Minify: JavaScript + CSS + HTML
  • Caching Level: Standard
  • Browser Cache TTL: 4 hours
